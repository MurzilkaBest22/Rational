ОПИСАНИЕ
Библиотека Rational предоставляет класс для работы с рациональными числами в виде несократимых дробей (числитель/знаменатель). Она также включает основные арифметические операции и математические функции для рациональных аргументов.

ВОЗМОЖНОСТИ
Класс Rational с конструкторами от целого числа или пары (числитель, знаменатель).
Приведение к несократимому виду (алгоритм Евклида).
Операции: +, -, *, /, составные присваивания, сравнения.
Методы toLongDouble() для преобразования в long double.
Функции: ln, lg, lb, pow, sqrt, cbrt, тригонометрические, гиперболические и обратные тригонометрические.
Проверка области допустимых значений (например, для логарифмов, корней чётной степени).

УСТАНОВКА И СБОРКА
Библиотека состоит из заголовочного файла rational.h и файла реализации rational.cpp. Для использования её в своём проекте:
Скопируйте папку Rational в ваш проект.
Добавьте rational.cpp к списку файлов компиляции.
Включите заголовок:

#include "Rational/rational.h"

Используйте пространство имён rational.
Если вы предпочитаете статическую библиотеку, скомпилируйте rational.cpp в Rational.lib и линкуйте её.
